### 참고 사항

- table에는 caption으로 반드시 테이블 설명을 제공해야 한다.
- 제목 셀과 내용 셀을 연결해야한다.
- 스크롤 테이블의 경우, 스크롤의 넓이가 inner table의 넓이에 포함되기 때문에 좌우 선이 있을 경우 넓이가 안맞을 수 있다.
