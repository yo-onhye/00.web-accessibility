고민중인 것
  • - 빈 태그를 사용
  • - 스크립트 사용
  • - 초점 잡히는 범위(?)
1. 라디오 커스텀 이미지를 빈 태그의 배경이미지로 처리할 경우
  • - 빈 태그를 넣어야해서 html 소스가 깔끔하지 않다.
  • - css의 checked를 사용하여 별도의 스크립트가 필요하지 않다. (ie 하위때문에 스킄립트를 넣긴.. 넣음)
  • - 초점이 inut요소에만 잡힌다.
2. 라디오 커스텀 이미지를 가상선택자로 처리할 경우
  • - 빈 태그를 넣을 필요가 없다. (불필요한 태그 사용 X)
  • - 스크립트를 사용했다. 처음 초기 checked도 스크립트로 해야되는데..
  • - label까지 통으로 초점이 잡힌다.
3. 라디오 커스텀 이미지를 감싼 태그 배경이미지로 처리할 경우
  • - 빈 태그를 넣을 필요가 없다. (불필요한 태그 사용 X)
  • - 레이블이 길어질 경우 스프라이트 이미지가 모두 노출 (배경이미지를 on/off 각각 잘라 사용하면 괜찮음)
  • - 왜 초점이 안잡히지ㅠㅠ
